# LearningPhraseApp

Web application for language learning. Focuses on regular repetition of words, phrases, and sentences. Can also be adapted for other types of educational material.<br/><br/>

## Table of Contents
- [Features](#features)
- [Technologies](#technologies)
- [Running the Application](#running-the-application)
- [Screenshots](#screenshots)
- [License](#license)

## Features
- User registration and login
- Adding and learning vocabulary
- Adding images
- Progress tracking


## Technologies
- Java 17
- Spring(Spring Boot, Spring Security, Spring MVC)
- Maven
- MS SQL Database
- Thymeleaf
- HTML
- CSS
- JavaScript
- Flyway
- JUnit 5
- Mockito

 ## Running the application

1. Clone the repository:
   ```bash
   git clone https://github.com/adamson32/LearningPhraseApp.git

    Navigate to the project directory:

    cd LearningPhraseApp

    Configure the database in application.properties

    Run the application:

    mvn spring-boot:run

    Open your browser and go to:

    http://localhost:8080
